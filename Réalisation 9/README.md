# Quiz PHP (sans base de données)

## Description
Petite application PHP qui permet de réaliser un quiz simple.
Le projet fonctionne en local, sans base de données et sans programmation orientée objet.

## Technologies utilisées
- PHP
- HTML

## Fonctionnement
1. L'utilisateur répond aux questions du quiz
2. Les réponses sont envoyées via POST
3. Le score est calculé et affiché

## Structure
- index.php : affichage du quiz
- result.php : calcul et affichage du score
- README.md : documentation

## Exécution
- Installer XAMPP ou WAMP
- Mettre le dossier dans `htdocs`
- Ouvrir : http://localhost/quiz-php
